# L'aventure de Stickman

## À propos du projet

Projet scolaire réalisé dans le cadre du cours Web 5 (582-511MO) au Cégep de Montmorency, programme Techniques d'intégration multimédia.

Trace ton chemin : "Cette histoire interactive propose une expérience narrative immersive où chaque choix du lecteur influence le déroulement et la conclusion de l'aventure.

## Description
L'aventure de stickman est un site interactif inspiré du jeu populaire donjon et dragons. Le frère du personnage principal se fait kidnapper par un méchant sorcier pour démarrer un rituel. Vous devez infiltrer la base du sorcier et faire les bon choix pour accumuler les outils nécessaires pour sauver votre frère du mal.

## Comment jouer
1. Cliquer sur le bouton "nouvelle partie" pour commencer l'histoire
2. Vous avez plusieurs choix pendant certains chapitres, vous devez donc cliquer sur un bouton de choix pour progresser.
3. Dans certains chapitres vous recevrez des objets. Ces objets peuvent être observés dans le menu de l'inventaire en cliquant sur le coffre. Cela puvre une page qui montre votre inventaire. Cliquez sur un objet pour recevoir la description de ce dernier.
4. À la fin, vous aurez le choix entre voir les statistiques de votre histoire et retourner à l'accueil immédiatement. En cliquant les statistiques, cela vous amènera à une page de statistiques qui montre tous les choix et les objets de votre histoire.
